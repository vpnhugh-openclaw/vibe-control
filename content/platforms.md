# Platforms

This is the working directory for where builds actually happen.

## What belongs here

- Platform strengths and weaknesses
- Best-fit use cases
- Which account should be used for each platform
- Credit posture, free-tier notes, and model access
- Practical guidance for where to start a project

## Suggested buckets

- **Builders:** Lovable, Base44, v0, Bolt
- **Code-first:** Cursor, VS Code + Copilot, GitHub Codespaces
- **Research / agentic:** Manus, Antigravity
- **Utility / fast experiments:** Replit, Firebase Studio

## Routing logic

- New UI-heavy web app: start at Lovable or v0
- Complex code refactor: move to Cursor or VS Code + Copilot
- Credit-constrained test: try Replit, Alt Dev, or a free-tier model
- Research-heavy workflow: try Manus first
