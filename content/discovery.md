# Discovery

Keep a running list of tools, promos, and shifts worth acting on.

## What belongs here

- New tools
- Free-tier or promo changes
- Community tips worth testing
- New workflow capabilities

## Triage rules

- **High relevance:** worth trying this week
- **Medium relevance:** bookmark and revisit
- **Low relevance:** ignore unless it supports an existing project

## Recommended actions

- Link useful discoveries to the most relevant project
- Re-check platform fit after meaningful platform changes
- Convert repeated discoveries into saved prompts or working playbooks
